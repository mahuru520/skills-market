# 故障排查

## 服务不可访问

1. 确认 ComfyUI 服务正在运行
2. 确认 `http://192.168.1.236:8188` 可访问

## 图片未先上传

视频生成前必须先上传输入图片，否则 `LoadImage` 无法找到文件。

## 队列长时间未完成

1. 先用 `/queue` 查看任务是否仍在执行
2. 1–2 分钟通常属于正常范围
3. 超时后检查模型或节点是否缺失

## 没有输出文件

1. 通过 `/history` 检查是否已有输出
2. 检查 `filename_prefix` 是否匹配
3. 检查 `subfolder=video&type=output` 是否正确

## 中文文件名下载问题

中文文件名建议使用 Python 方式下载，以处理 URL 编码。

## 质量不理想

1. 输入图尽量清晰
2. 正向提示词优先使用英文动作描述
3. 保持默认负向提示词
