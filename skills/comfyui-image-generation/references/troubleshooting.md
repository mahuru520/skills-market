# 故障排查

## 服务不可访问

1. 确认 ComfyUI 服务正在运行
2. 确认 `http://192.168.1.236:8188` 可访问

## 队列卡住或任务长时间不完成

1. 使用 `/queue` 检查是否还有未完成任务
2. 确认工作流依赖的模型和节点已安装

## 没有输出文件

1. 使用 `/history` 确认是否存在输出记录
2. 检查 `filename_prefix` 是否正确

## 节点或模型问题

可进一步检查：

```bash
curl -s "http://192.168.1.236:8188/object_info"
curl -s "http://192.168.1.236:8188/history"
```

## 图片质量不理想

1. 优先使用英文提示词
2. steps 调整到 25–35
3. cfg 保持在 4–7
4. 更换 seed 生成不同变体
