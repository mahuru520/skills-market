# 飞书插件能力模块

安装飞书插件后，系统将自动加载以下能力模块：

## 核心模块

| 模块 | 功能说明 |
|------|----------|
| 消息接收与回复 | 接收飞书消息并通过 AI 回复 |
| 上下文管理 | 维护对话上下文，支持多轮对话 |
| 工具调用 | 调用 OpenClaw 工具能力 |
| 文件处理 | 处理上传的文档、图片等 |

## 高级功能

| 模块 | 功能说明 |
|------|----------|
| 群聊管理 | 支持群@、群聊独立上下文 |
| 线程模式 | 每条消息独立线程会话 |
| 流式响应 | 流式输出 AI 回复 |
| 卡片消息 | 支持富文本卡片交互 |

## 配置能力

| 配置项 | 说明 |
|--------|------|
| requireMention | 仅@时回复 / 回复所有 |
| threadSession | 线程独立上下文 |
| streaming | 流式响应 |
| footer | 卡片底部信息 |

## 技能扩展

飞书插件可配合其他技能使用：

- `minimax-docx` — 文档处理
- `minimax-pdf` — PDF 生成
- `md-to-pdf` — Markdown 转 PDF
- `comfyui-image-generation` — 图片生成

## 使用示例

### 基础对话

```
用户：你好
AI：你好！有什么可以帮助你的？
```

### 工具调用

```
用户：帮我把这个文档转成 PDF
AI：[调用工具处理]
```

### 群聊交互

```
@AI 帮助解答这个问题
AI：[基于上下文回复]
```
