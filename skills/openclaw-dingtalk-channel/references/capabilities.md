# 钉钉连接器能力说明

安装钉钉连接器后，可获得以下能力：

## 核心能力

| 能力 | 说明 |
|------|------|
| AI Card 流式响应 | 实时展示 AI 回复 |
| 会话持久化 | 多轮对话共享上下文 |
| 会话与记忆隔离 | 单聊、群聊、不同群之间上下文隔离 |
| 手动新会话 | 支持 `/new`、`/reset`、`/clear` |

## 富媒体能力

| 能力 | 说明 |
|------|------|
| 图片接收 | 自动下载并传递给视觉模型 |
| 文件附件提取 | 支持 `.docx`、`.pdf`、`.txt`、`.md` 等 |
| 音频消息支持 | 支持多种音频格式 |
| 主动发送消息 | 可主动向个人或群发送消息 |

## 高级能力

| 能力 | 说明 |
|------|------|
| 多 Agent 路由 | 不同钉钉机器人可绑定不同 Agent |
| 异步模式 | 立即回执，后台处理并推送结果 |
| Markdown 表格转换 | 优化钉钉消息可读性 |
| 钉钉文档 API | 创建、追加、搜索、列举文档 |

## 典型场景

- 钉钉群聊 AI 助手
- 企业内部知识问答机器人
- 多角色、多 Agent 专业服务入口
- 文档、图片、音频混合处理场景
